- 📊 **7种图表类型** — 柱状图、折线图、饼图、面积图、环形图、散点图、雷达图
- 🎨 **6种精美配色** — 从专业商务到活力彩色，一键切换
- 📋 **粘贴即用** — 直接粘贴Excel/CSV数据，自动解析
- 💾 **高清导出** — 导出PNG图片，适合报告和演示
- ✏️ **实时编辑** — 所见即所得，调整即时生效
- 📱 **响应式设计** — 手机电脑都能用

## 🚀 使用方式

1. 打开 [ChartSnap](https://tyr1105.github.io/chartsnap/)
2. 在数据区粘贴你的数据（支持Excel直接粘贴）
3. 选择图表类型和配色
4. 点击导出，高清PNG到手
